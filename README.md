# gurongi_padrino

# How TO
## set up
### 1. clone
`git clone git@github.com:UltraBirdTech/gurongi_padrino.git`

### 2. cd 
`cd ./gurongi_padrino`

### 3. sqlite3
`brew install sqlite3`
or 
`apt-get install libsqlte3-dev`

### 4. bundle install
`bundle install`

## start
1. start up command
`bundle exec padrino start`

2. connect to URL
127.0.0.1:3000/translate

## stop
`bundle exec padrino stop`
